# POKEXPERTO - ANGULAR SPA CON GSAP

## Instalacion rapida

```bash
git clone https://github.com/MoneX2F/pokexperto-ng.git
cd pokexperto-ng
npm install
npm start
```

## Dependencias
- Angular 17
- - TypeScript 5
  - - GSAP 3.x
    - - RxJS 7
      - - SCSS
       
        - ## Estructura
       
        - ```
          src/
          ├── app/
          │   ├── core/          # Servicios globales
          │   ├── shared/        # Componentes compartidos
          │   ├── features/      # Modulos de features
          │   └── styles/        # Estilos SCSS
          ├── assets/
          ├── index.html
          └── main.ts
          ```

          ## Comandos

          - `npm start` - Desarrollo
          - - `npm run build:prod` - Produccion
            - - `npm test` - Tests
              - - `ng generate component path/name` - Crear componente
                - - `ng generate service path/name` - Crear servicio
                 
                  - ## Demo
                 
                  - Abre `/public/pokexperto-demo.html` en tu navegador para ver tarjetas Pokemon interactivas.
